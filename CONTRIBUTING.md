# Contributing

Thanks for helping build Open Agent Access for Algorand.

## Development

```sh
corepack enable
pnpm install
pnpm build
pnpm test
pnpm lint
```

Keep changes small, typed, tested, and documented. Do not add real wallet
addresses, mnemonics, private keys, seed phrases, or production secrets.
