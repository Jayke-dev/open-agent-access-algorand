# Kirke Labs Stewardship

Open Agent Access for Algorand is maintained as a Kirke Labs open infrastructure
project.

The project goal is not to own agent access, but to make reciprocal access,
payment, and receipts practical enough that agents, publishers, APIs, and tool
owners can interoperate without fragile scraping norms.

Stewardship principles:

- open protocol before closed platform
- Algorand-first without payment-adapter lock-in
- builders get practical SDKs and middleware, not only specs
- resource owners get rate, load, payment, and receipt controls
- security posture is documented, tested, and enforceable in CI
