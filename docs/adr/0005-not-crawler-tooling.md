# ADR 0005: Not Crawler Tooling

## Status

Accepted.

## Context

Agent access infrastructure can be confused with scraping automation.

## Decision

Open Agent Access implements discovery, decisions, rate/load/payment terms, and
receipts. It does not implement crawling, scraping, bypassing, or paywall
avoidance.

## Consequences

The project remains aligned with reciprocal access and resource owner control.
